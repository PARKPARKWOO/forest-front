## 슬라이드 3: 데이터 이관 기능

# Q1. 데이터 이관 기능이 있나요?

## ✅ 네, 있습니다!

**사용 목적**
- 셀픽으로 등록하지 않은 상품을 셀픽에서 관리하고자 할 때 사용

**사용 방법**
1. [쇼핑몰 상품 가져오기]로 상품 등록
2. [쇼핑몰 상품데이터 이관]에서 코드 매칭
3. 연동상품코드 생성 및 연결

---

## 슬라이드 4: 데이터 이관 상세 과정

# 데이터 이관 상세 과정

## Step 1: 상품 등록
- [쇼핑몰 상품 가져오기] 메뉴
- 쇼핑몰에 등록된 상품과 같은 상품을 셀픽에 등록
- 단품 및 상품-단품등록, 상품등록

## Step 2: 코드 매칭
- [쇼핑몰 상품데이터 이관] 메뉴
- 셀픽의 상품코드(또는 판매자상품코드)에 해당하는 쇼핑몰의 상품코드 입력

## Step 3: 저장 및 연결
- 매칭할 쇼핑몰 선택 후 저장
- 연동상품코드 생성
- 연동상품코드 ↔ 쇼핑몰 상품코드 연결 완료

---

## 슬라이드 5: 데이터 이관 주의사항

# ⚠️ 데이터 이관 주의사항

## 중요 안내

**데이터 이관의 본질**
- 단순 코드의 연결입니다
- 데이터 100% 동기화가 아닙니다

**데이터 동기화**
- [쇼핑몰 상품조회/수정]에서 데이터 동기화를 통해 쇼핑몰 데이터를 가져옵니다
- 수정 전송 전 **연동상품코드를 반드시 확인**하세요

## 슬라이드 6: 일괄 변경 기능

# Q2. 쇼핑몰 등록상품이 일괄변경될 수 있나요?

## ✅ 네, 가능합니다!

**일괄 변경 가능 항목**
- 기본정보
- 배송정보
- 카테고리

**사용 방법**
1. [쇼핑몰 상품조회/수정] - [상품조회]
2. 변경할 상품 선택
3. [일괄 수정] 클릭
4. 기본/배송/카테고리 선택 후 변경
5. 저장 후 [쇼핑몰전송]

---

## 슬라이드 7: 일괄 변경 상세 과정

# 일괄 변경 상세 과정

```
[쇼핑몰 상품조회/수정]
    ↓
[상품조회]
    ↓
상품 선택 (체크박스)
    ↓
[일괄 수정] 버튼 클릭
    ↓
변경 항목 선택
  - 기본정보
  - 배송정보
  - 카테고리
    ↓
변경 내용 입력 후 저장
    ↓
[쇼핑몰전송] 클릭
```

## 슬라이드 12: 기존 상품 불러오기

**사용 조건**
- 쇼핑몰 계정 연동 완료

**사용 방법**
1. [쇼핑몰] - [쇼핑몰연동관리]
2. [쇼핑몰 상품 가져오기 / 기능]
3. 셀픽으로 상품 불러오기

**장점**
- 기존 상품 정보 자동 수집
- 수동 입력 불필요
- 빠른 상품 등록
